Phaser 3 API Documentation - Namespace: In

叠置对齐（同侧对齐）。 另有Phaser.Display.Align.To类，使两个对象并置对齐（对侧对齐），方法与此近似。

Phaser.Display.Align. In

Methods

BottomCenter(gameObject, alignIn [, offsetX] [, offsetY])

gameObject与alignIn两个游戏对象，叠置时底边中心对齐（同侧对齐）。以下各项类此。
