派真与两条腿的哈罗世界

2020-12-06 周日 by 黄复雄.

准备写一个Python的入门教程，给小学三年的孩子和心境相似的成人。现在只是不成文的思路。

创世纪

Python（派真，一条漂亮的小蟒蛇）：传说，是神创造了世界。星期一……亚当、肋骨、夏娃、蛇、苹果……女娲…… TwoLegs（两条腿，一只貌似憨厚的猿）： 派真：那么在神创造世界之前有什么呢？当然只有神。也就是说，创造世界之前首先要有创造神界的工具，神的工具就是他自己，具体来讲就是语言：神说“要有光”，于是就有了光。

哈罗世界/Hello World！

派真：那么我们也来创造一个世界，就是程序猿们最喜欢的哈罗世界(HelloWorld!)。我的工具就是我，也就是我的语言——Python。

【安装Python】

两条腿的第一个魔法

print("Hello World")

变量/盒子还是盒子标签

各从其类/all in class

派真：神造物不是一个个造的，是一类一类地造。也就是“各从其类。”

# 类
class HelloWorld:
    print("Hello World")

# 用类生成的实例
helloWorld = HelloWorld()
helloThere = HelloWorld()

print(HelloWorld)
print(helloWorld)
print(helloThere)

神按自己的样子造人/类的继承

class God:
    pass

class Python(God):
    pass

class TwoLegs(God):
    pass

class Eve(TwoLegs):
    pass

全都是对象/基类/object
