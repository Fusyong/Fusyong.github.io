Python找门教程-前戏-搭建开发环境

2020-12-06 周日 by 黄复雄.

准备写一个最简单的Python教程，最简单的那种，就是让读者在入门之前，先找着了门。给小学三年的孩子和心境相当的中老年人。现在只是不成文的思路。

编程，编什么编？在哪里编？

【微信聊天界面。派真发海报：少儿不Python，老大学编程。配潘石屹图】

【两条腿】 老铁，海报要正能量啊，给客户正向感受。 改成：少儿学Python，一家会编程

【派真】 OK，少儿学Python，一家跳大绳 闲话少说，言归正传。 今天第一堂课，我们来讲：编程，编什么编？在哪里编？

【两条腿】 我要学Python数据分析。

【派真】 中年人，别心急。 先把你的Python安装好。根据你电脑的操作系统，选择下面的Python下载链接，打开，下载，安装。如果是Windows系统，一定要勾选Add Python 3.x to PATH选项。

  • Windows
  • Linux/UNIX
  • Mac OS X
  • Other

【派真】 再到这里下载微软的宇宙第一编辑器VS Code，一路选按默认选项安装。

如果需要中文界面，这样安装中文插件： 【待实测】

【两条腿】 看起来很难。

【派真】 找不着门的时候就是这种感觉，没头的苍蝇似的。我们这个课程就是帮你找到门。然后你就可以自己进去了。

现在我们在Code里面新建一个文件（ctrl+n），在里面下来你的第一个魔法（这是程序员们伟大且无聊的传统）：

print("Hello World!")

请分清全角和半角标点符号，长得像不等于一样。

然后保存文件（ctrl+s），名字为hello_world.py，后缀名不能弄错。

这是会提示你安装【需实测】插件，一路默认安装。

安装好后点击右上角的三角形按钮，运行你的代码。你会看到一个终端打开了，并有类似这样的文字输入来：

Hello World!

恭喜，你的第一个脚本成功运行了！

【两条腿】 啊，这样啊。看起来挺简单。 另外，脚本是什么？

【派真】 没找着门时的另一种感觉/错觉就是“看起来挺简单”。 解释几个词语吧。

变量/盒子还是盒子标签

各从其类/all in class

派真：神造物不是一个个造的，是一类一类地造。也就是“各从其类。”

# 类
class HelloWorld:
    print("Hello World")

# 用类生成的实例
helloWorld = HelloWorld()
helloThere = HelloWorld()

print(HelloWorld)
print(helloWorld)
print(helloThere)

神按自己的样子造人/类的继承

class God:
    pass

class Python(God):
    pass

class TwoLegs(God):
    pass

class Eve(TwoLegs):
    pass

全都是对象/基类/object
